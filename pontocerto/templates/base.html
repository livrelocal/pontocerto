Sobre o Ponto Certo

Ponto Certo é um projeto de pesquisa com o objetivo é dar visibilidade às condições de acessibilidades dos pontos de ônibus de Salvador através de dados georeferenciados, coletados a partir de inspeções técnicas realizadas in loco e em parceria com o CREA-BA. Este projeto é financiado pelo CNPQ.

Este é um projeto Software Livre sob a Licença AGPL. O seu código encontra-se disponível neste repositório do GitHub.

Para fazer o download do geojson, clique aqui. Os dados estão disponíveis sob a Licença ODBL e seus créditos, além dos do mapa, são do Open Street Map.
